# Audio

This folder contains audio files for the game.

## Supported Formats

Unity supports various audio formats:
- WAV (uncompressed, best quality)
- MP3 (compressed, smaller size)
- OGG (compressed, good balance)
- AIFF

## Audio Organization

Consider organizing audio files into subfolders:
- Music/ - Background music tracks
- SFX/ - Sound effects
- Voice/ - Voice acting and dialogue
- Ambient/ - Ambient sounds

## Best Practices

- Use WAV for short sound effects
- Use compressed formats (MP3/OGG) for longer tracks
- Keep audio files at appropriate sample rates (44.1kHz is standard)

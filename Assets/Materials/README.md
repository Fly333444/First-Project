# Materials

This folder contains Unity material files.

## About Materials

Materials define how surfaces look in your game. They control:

- Color and texture
- Reflectivity and metallicness
- Smoothness and roughness
- Emission and transparency

## Creating a Material

1. Right-click in this folder
2. Select Create > Material
3. Assign textures and adjust properties in the Inspector
4. Drag the material onto objects in your scene

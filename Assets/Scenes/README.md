# Scenes

This folder contains Unity scene files for the game.

## Creating Your First Scene

1. Open Unity Editor
2. Go to File > New Scene
3. Save the scene in this folder (Assets/Scenes/)
4. Build your game world using GameObjects, lights, and cameras

## Scene Organization

- Keep scene files organized by level or game area
- Use descriptive names for scenes
- Consider creating a main menu scene and gameplay scenes separately
